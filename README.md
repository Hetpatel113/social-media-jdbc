# Social Media App (Java + JDBC)

This is a console-based social media application built using Java, JDBC, and MySQL/MariaDB.

## Features
- User register & login
- Create posts
- Like, comment, share
- Follow system
- Notifications using SQL triggers
- Profile with bio
- Private accounts
- Image upload support (BLOB)

## How to Setup Database
1. Create database:CREATE DATABASE social_media;

2. Import SQL file:
- Open phpMyAdmin → select `social_media` database → go to Import → upload `sql/schema.sql`
